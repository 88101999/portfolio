## サービス概要
コーディネートの提案を行うサービスです。
自分が持っている洋服や、欲しいなあと思っている洋服を登録すると、
そのアイテムに合うようなコーディネートを提案し、
ユーザーは気に入ったものを保存することができます。

## このサービスへの思い・作りたい理由
前職でアパレル店員として働いており、多くのお客様のコーディネートに関する悩みを聞いてきました。
その中で、「自分に似合う洋服がわからない」「洋服の合わせ方がわからない」といった悩みを抱えている方が多いことに気づきました。
また、話すのが面倒といった理由や、押し売りされそうという理由で、接客に苦手意識のあるお客様が多いことも実感しました。

そこで、わざわざ店舗に行ってスタッフと喋らなくても
自分が持っている洋服や欲しい洋服を登録するだけで、そのアイテムに合うコーディネートを提案してくれるサービスがあれば、
上記の悩みを解決できるのではないかと考え、このサービスを作ろうと思いました。

## ユーザー層について
20代~30代の男女を主なターゲットユーザーとしています。
特に、ファッションに興味はあるけれども、どんな服がいいかわからない、
わざわざ店舗に行って接客を受けるのは面倒だと感じている方々を想定しています。

## サービスの利用イメージ
ユーザーはまず、自分が持っている洋服や欲しい洋服の情報をプルダウンから選択する形でサービスに登録します。
プルダウンには、アイテムの種類（トップス、ボトムス、シューズなど）や色などが登録されています。

その後、サービスは登録されたアイテムに基づいて、複数のコーディネートパターンを提案します。
ユーザーは提案されたコーディネートの中から気に入ったものを選び、保存することができます。
保存したコーディネートは後で見返すことができ、日々の洋服選びの参考にすることができます。

## コーディネートの提案方法
データベースに事前に20〜30パターンの組み合わせを登録しておき、
ユーザーが登録したアイテムに沿ってコーディネートを数パターン表示する形を想定しています。

## ユーザーの獲得について
- SNSでの発信: InstagramやXなどファッションに興味がある層が多いプラットフォームで、実際のコーディネート提案例を投稿しながらサービスを紹介します。
- 知人への紹介: 身近な友人や、前職のアパレル店員時代の知人に使ってもらい、フィードバックをもらいながらサービスを改善していきます。

## サービスの差別化ポイント・推しポイント
Google検索やInstagram、Pinterest、またはファッション雑誌等でコーディネートを調べることもできますが、
これらの方法は、自分が実際に持っている洋服を軸にコーディネートを提案してくれるわけではありません。
このサービスでは、ユーザーが登録した洋服に合うコーディネートを提案することで、ユーザーの悩みを解決できるのではないかと考えています。

## 機能候補
### MVPリリース時に作っていたいもの
- アイテム登録機能: ユーザーが自分の持っている洋服や欲しい洋服を登録できる機能(アイテムはトップス、アウター、ボトムス、シューズの4種類、色は白系、黒系、寒色系、暖色系の4種類に限定)
- コーディネート提案機能: 登録されたアイテムに基づいて、複数のコーディネートパターンをテキストで提案する機能
- お気に入り保存機能: ユーザーが気に入ったコーディネートを保存・表示できる機能

### 本リリースまでに作っていたいもの
- アイテム登録機能: トップス:Tシャツ、シャツ/アウター：ジャケット、コート等/ボトムス:デニム、チノパン、スラックス、スカート等/シューズ:スニーカー、革靴、パンプス等サブカテゴリを追加。色:基本的な色味10色程度に増加
- コーディネート提案機能: イラスト付きでの提案。「カジュアル」、「キレイめ」などジャンル別の提案。
- UI/UX改善: ユーザビリティの向上、デザインの改善(具体的な内容はまだ定まっておりません。)
